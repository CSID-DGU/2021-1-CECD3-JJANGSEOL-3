# 2021-1-CECD3-JJANGSEOL-3
### IoT sensor data 수집 및 CNN 활용을 통한 독거노인 우울증상 초기감지 with 고그린테크

## 짱설팀
* 이미란 (팀장)
* 노승수
* 박지호
* 박희상

독거노인 응급안전안심서비스 차세대 댁내 장비 사용하여 센서데이터 수집 후 그래프로 시각화
